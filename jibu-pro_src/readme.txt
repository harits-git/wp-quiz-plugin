Jibu Pro: the WordPress exam system.

== Description ==

Jibu Pro is a WordPress plugin that enables you to create quizzes and tests on your WordPress website. Your website visitors will then be able to take these tests online on your site and view results afterwards.

This allows you to:

*   Engage your site visitors in a creative and fun way
*   Create an online learning environment on your website
*   Offer online courses and have your learners take actual tests online
*   Create a CBT (computer based training) solution on your website or for your company/school

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
